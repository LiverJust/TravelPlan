🌿 道東（Dōtō）

北海道最原始的大自然秘境——知床世界遺產、神秘道東三湖（摩周湖・屈斜路湖・阿寒湖）、硫磺山活火山、帶廣酒造，是日本最壯美的自然區域。

🌸 道北（Dōhoku）

北海道北部花田王國——層雲峡壯觀峽谷、旭山動物園、美瑛青池、富良野薰衣草花海（特別增加2晚深度遊！），加上三井Outlet大型購物圓滿收尾！

👨‍👩‍👧‍👦 家庭成員 6人

  • ✈抵達新千歲機場（CTS），辦理入境手續、提取行李

  • 🍛晚餐：帶廣「インデアンカレー」——在地人從小吃到大的咖喱連鎖，物美價廉，適合抵達後輕鬆用餐

Day 2
7/21
🚂 帶廣 → 弟子屈｜幸福駅・酒造・拉麵
幸福車站 → 豚丼一番（Lunch）→ 碧雲蔵 → 弟子屈拉麵（Dinner）→ 屈斜路王子大飯店
道東 ▼
  • 🚂舊幸福車站（旧幸福駅）：帶廣近郊約15km。昔日廣尾線的廢棄鐵路站，「愛国から幸福へ」（從愛國到幸福）是昭和時代日本最著名的車票，大量情侶來此貼車票祈願幸福。站旁有小型展示館，環境清靜適合早上慢逛
  • 🍖Lunch @ 豚丼一番（帶廣市內）：帶廣豚丼名店！精選十勝豬肉，炭火慢烤配秘製醬汁，每口都是濃郁肉香。店內有中文菜單，推薦「特上豚丼」（¥1,500 左右），排隊是值得的！
  • 🍶上川大雪酒造 碧雲蔵 Shop（帶廣）：北海道頂級地酒！碧雲蔵是上川大雪酒造設於帶廣的釀造所兼直銷店，使用十勝山脈伏流水和北海道本地酒米釀製。店內可試飲多款季節限定純米酒，購買限定版清酒帶回家最合適！
  • 🛣下午出發，沿国道 38 號向東行，途經 足寄（あしょろ），抵達弟子屈。全程約 130km，約 2 小時
  • 🍜Dinner @ 弟子屈拉麵 総本店：道東最著名的拉麵！濃郁魚介湯底，麵條爽滑，是整個道東行的必吃美食。就在弟子屈市中心，晚餐到達後直接前往
🍶 碧雲蔵小知識：上川大雪酒造創立於2018年，是北海道近年最受矚目的地酒品牌。碧雲蔵（帶廣）與本店（層雲峡附近上川町）均可購買，但限定款式各有不同。7月有夏季限定「吟醸生酒」和「純米大吟醸」，建議購入2-3瓶作伴手禮！
💡 帶廣出發前可在市區超市採購零食和飲料。弟子屈的屈斜路王子大飯店直接位於屈斜路湖畔，入住後可享受湖景！
♿ 長者友善評估：舊幸福車站廣場平坦，長者可輕鬆散步。碧雲蔵 Shop 室內購物，全程無須步行遠路。帶廣→弟子屈途中建議在足寄道之駅停車休息一次（約1小時後）。
🏨 今晚住宿：屈斜路プリンスホテル（屈斜路王子大飯店）——直接位於屈斜路湖畔，有大浴場，湖景絕美。地址：北海道川上郡弟子屈町屈斜路
Day 3
7/22
🌟 道東精華大日！硫黃山・摩周湖・阿寒湖・愛努族
弟子屈 → 硫黃山 → 摩周湖 → 屈斜路湖砂湯 → 北海道夢幻橋 → 阿寒湖 → 愛努族民俗村 → 返回弟子屈
道東 ▼
  • 🌋硫黃山（Iozan）：仍在活動的活火山！硫磺煙霧不斷噴出，地面滾燙，近距離感受大地的力量。旁邊商家用地熱烤玉米和雞蛋——必吃！免費入場，停車¥500
  • 💙摩周湖第一展望台：世界透明度最高的湖泊之一，湖水深藍神秘，清早有可能見到「霧の摩周湖」，收費停車場¥500
  • 🔭摩周湖第三展望台（推薦！）：比第一展望台更震撼的全景視野，可同時看見摩周湖全貌及知床連峰，免費停車，人更少！
  • 🏖屈斜路湖 砂湯：湖邊沙灘下湧出天然溫泉，用手挖開沙子即可免費泡腳！傍晚夕陽映照屈斜路湖，景色絕美。酒店就在附近
  • 🌉北海道夢幻橋（途中必停！）：弟子屈往阿寒湖路上的秘境，木橋橫跨碧綠溪流，被茂密原始森林包圍，是北海道隱藏版絕景，停留約20分鐘
  • 🪶阿寒湖愛努族民俗村（アイヌコタン）：日本最大愛努族聚落，有傳統舞蹈表演「イコロ」（每晚定時演出，入場費¥1,080）、手工藝品店、傳統食品，深度了解北海道原住民文化
  • ⛵阿寒湖遊覧船：乘坐觀光船環遊阿寒湖，途中參觀「まりも（毬藻）」展示——世界稀有的球狀藻類，是阿寒湖的象徵（視時間決定是否參加）
  • 🚗晚上從阿寒湖返回弟子屈（約50km，約45分鐘），回到酒店享受屈斜路湖露天溫泉
💡 行程安排建議：早上8:00出發，先走硫黃山→摩周湖（上午）→砂湯（中午）→夢幻橋（下午）→阿寒湖愛努族民俗村（傍晚-晚上）→返回弟子屈。充實但精彩的一天！愛努族舞蹈通常19:00或20:00演出，看完即可返程。
♿ 長者友善：硫黃山停車場到火山口步行僅5分鐘，平坦。摩周湖第三展望台停車場直達，完全適合長者。砂湯沙灘柔軟，長者坐椅子泡腳即可。阿寒湖民俗村地面平坦，愛努族舞蹈在室內劇場有座椅。
🏨 今晚住宿：屈斜路プリンスホテル（連住第2晚）——日間精華遊覽阿寒湖後返回弟子屈過夜，免去搬行李之苦！

7/23
🦅 弟子屈 → 知床｜世界遺產入境
弟子屈/屈斜路 → 中標津 → 斜里 → 知床宇登呂（約120km / 約2小時）
道東 ▼
  • 🛣早上退房，從弟子屈出發，沿国道 243 → 中標津 → 国道 244 → 斜里 → 知床宇登呂。全程約 120km，約 2 小時，建議在中標津道之駅停車休息
  • 🌲知床五湖（Shiretoko Goko）：UNESCO 世界自然遺產！5個湛藍湖泊隱藏於原始森林，7月開放「地上步道自由散步」，沿木棧道漫步，極有可能近距離見到棕熊！
  • ⛵知床観光船（ゴジラ岩観光）：乘船沿知床半島海岸航行，欣賞奇岩峭壁、瀑布直入大海、野生棕熊在岸邊覓食——整個行程最難忘的體驗！提前網上預訂
  • 🌅知床峠（初見）：如時間允許，傍晚可駕車上知床峠欣賞日落，晴天可望見俄羅斯領土國後島
📍 路線優化：先知床→後網走，知床→網走僅70km（1小時），大幅縮短後段路程！
💡 知床五湖重要提醒：7月開放「地上步道自由散步」，需在入口處登記並觀看5分鐘熊出沒說明影片。可在現場租借防熊噴霧。
♿ 長者友善方案：知床五湖地上步道全程3km，對75歲以上長者較吃力。建議長者走「高架木道（800m、平坦、無需登記）」，從1號湖展望台遠眺即景色絕美。年輕成員走全程後在入口廣場匯合。
🏨 今晚住宿：宇登呂温泉（推薦 知床グランドホテル北こぶし——有電梯、無障礙客房，長者首選）
Day 5
7/24
⭐ 知床深度遊｜羅臼・觀鯨・觀星
宇登呂 → 知床峠 → 羅臼 → 觀鯨船 → 深夜觀星
道東 ▼
  • 🛣知床峠ドライブ：清早翻越知床峠（738m），晴天可望見俄羅斯領土國後島，是北海道最美的高山駕駛路段
  • 🐳羅臼 觀鯨船：知床半島東側小鎮羅臼，夏季有深海觀鯨船出發，可能看到抹香鯨！推薦大型船（室內座位 + 廁所，適合長者），建議早上出發
  • 🦅羅臼岳ビジターセンター：棕熊、虎頭海鷹標本展示，了解知床國立公園生態系統，全家適合
  • 🍣晚餐：羅臼港邊海鮮丼——當地現撈！比市區便宜，新鮮度極高，7月海膽正當季
  • ⭐深夜觀星：知床遠離城市光污染，夏夜晴天可見銀河橫跨天際，北海道最佳觀星地點之一
♿ 長者友善：羅臼觀鯨大型船有室內座位和廁所，長者可在室內欣賞。觀星如長者怕冷，可在酒店陽台觀星，同樣效果極佳。
🏨 今晚住宿：宇登呂温泉（連住2晚）

7/25
🚔 知床 → 網走｜監獄・流氷館・鄂霍次克
知床宇登呂 → 網走（約70km / 約1小時）→ 輕鬆的移動日！
道東 ▼
  • 🛣早上悠閒退房，沿国道 334 → 斜里 → 国道 244 前往網走，全程僅約 70km，約1小時——是全程最輕鬆的移動！充裕時間遊覽網走
  • ⛓網走監獄博物館（博物館網走監獄）：日本最有名的舊監獄開放參觀！明治時代木造監舍群完整保留，蠟像人偶重現囚犯生活，大人小孩都大開眼界。需2-3小時
  • ❄鄂霍次克流氷館（オホーツク流氷館）：展示真實流冰！室內溫度-15°C，夏天穿薄衣服進去體驗冬天，可觸摸真正流冰，還有可愛的流冰天使（クリオネ）展示
  • 🏔天都山展望台：俯瞰鄂霍次克海、網走湖、能取湖的360度全景，天氣好時視野極佳
  • 🦀晚餐：網走港海鮮——毛蟹、鮭魚、海膽，7月正是新鮮海膽的旺季！
✅ 路線優化：知床→網走僅70km（1小時），比舊路線節省約200km車程！
💡 7月網走平均氣溫僅18°C，非常舒適！進入流氷館時提供租借外套，長者進去前先穿好防寒衣物。
♿ 長者友善：流氷館有升降機，長者方便參觀。監獄博物館建議乘坐館內電動車（有收費接駁服務），節省長者體力。
🏨 今晚住宿：網走市（推薦 東橫Inn 網走站前 或 ホテルルートイン網走）

7/26
🏔 網走 → 層雲峡｜大峽谷・銀瀑・溫泉
網走 → 北見 → 遠軽 → 層雲峡（約140km / 約2小時）
大雪山 ▼
  • 🛣早上退房，沿国道 39 號西行：網走 → 北見（~44km）→ 遠軽（~50km）→ 上川 → 層雲峡（~46km），全程約140km，約2小時。途中可在北見道之駅補充食物
  • 🌊大函・小函（Obako / Kobako）：層雲峡最壯觀的柱狀節理峽谷！數十萬年前火山噴發形成的六角形玄武岩柱壁高達100-200m，氣勢磅礴。步道平坦，沿峽谷漫步約30分鐘
  • 💦銀河の滝・流星の滝：層雲峡最著名的雙瀑布！銀河の滝纖細飄逸如銀河傾瀉，流星の滝豪邁奔放如流星墜落。從停車場步行僅約5分鐘，免費，景色震撼！
  • 🚡黒岳ロープウェイ：乘纜車登上黒岳（1520m），遠眺大雪山連峰壯觀全景，7月山頂仍可見殘雪，夏日雪景別具一格
  • ♨層雲峡温泉：在大峽谷旁泡溫泉，夜間溫泉旅館提供豐盛北海道會席料理
💡 層雲峡是大雪山國立公園的門戶，夏天峽谷翠綠、瀑布澎湃，完全不遜色秋季紅葉！銀河・流星雙瀑是全程最好拍照的自然景點之一。
♿ 長者友善：大函・小函步道平坦，長者可慢步欣賞。雙瀑布距停車場僅5分鐘，完全適合長者。黒岳纜車有座位，上山後在觀景台休息眺望，無需步行登頂。
🏨 今晚住宿：層雲峡温泉（推薦 層雲峡観光ホテル 或 朝陽リゾートホテル，有電梯・無障礙房・溫泉大浴場）

7/27
🐧 層雲峡 → 旭川｜旭山動物園
層雲峡 → 旭川（約65km / 約1小時）→ 旭山動物園
道北 ▼
  • 🛣早上悠閒退房，沿国道 39 號南下前往旭川，全程約65km，只需1小時，是全程最輕鬆的移動日之一
  • 🐧旭山動物園（Asahiyama Zoo）：日本最著名的動物園！以「行動展示」方式讓動物展現自然行為——企鵝空中飛翔、北極熊水中游泳、猩猩空中穿梭。可安排整個下午
  • 🦁精華展館：企鵝館・北極熊館・猩猩館・雪豹館，每館各需20-30分鐘。夏季有夜間開放！門票大人¥820，中學生以下免費
  • 🍺晚餐：旭川ラーメン——濃厚醬油・豚骨雙湯底，推薦「蜂屋」（創業70年）或「青葉」
♿ 長者友善：旭山動物園提供免費輪椅借用（押金¥500），園區地面硬化平整。建議下午2點後入場，避開早上人流，天氣涼快時體驗更佳。
🏨 今晚住宿：旭川市（推薦 JRイン旭川 或 ドーミーイン旭川——車站附近，電梯齊全）

🌸 美瑛・富良野｜薰衣草花海盛開！
